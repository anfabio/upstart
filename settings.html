Import & Export

Importação & Exportação

Here you can import and export you data and settings as well as reset them. You can also import the browser bookmarks which will create pages and groups to acommodate them. You can also choose what will be imported. It is worth mentioning that there is an option to try to fetch the bookmark icon, which can take a reasonable time depending of the number of bookmarks. If you find any problem with this import, please uncheck this option and try again. Aqui você pode importar e exportar seus dados e configurações além de redefini-los. Você também pode importar os favoritos do navegador o que criará páginas e grupos para acomodá-los. Você também poderá escolher o que será importado. Vale lembrar que há uma opção para tentar obter os ícones de cada favorito, o que pode levar um tempo razoável dependendo da quantidade. Caso você encontre algum problema nesta importação, desmarque esta opção e tente novamente.

Convert Icons to base64 data:image

Converter Ícones para base64 data:image

This action will convert the bookmarks icons to base64 data:image (webp/128x128 pixels). After this, the icons will be stored within the data itself and will not be loaded from a file or URL every time the page loads.
The main drawback is the fact that the data size will be bigger. This could make the pages load a bit slower depending on the number of bookmarks you have. Besides, the icons will be stored with a resolution of 128x128 pixels, so they will loose some details if you prefer really bigger ones (the default icons are shown as 64x64 pixels). Be sure to export your data or create a restore point first, so this can be reverted. There is also a option to convert each icon individually at the bookmark menu Esta ação vai converter todos os ícones dos favoritos para base64 data:image (webp/128x128 pixels). Depois disso, os ícones serão armazenados dentro dos dados em si e não serão carregados de um arquivo ou URL toda vez que a página carregar.
O único inconveninete é o fato de que o tamanho dos dados será maior. Isto pode deixar o carregamento das páginas um pouco mais lento dependendo da quantidade de favoritos. Além disso, os ícones serão guardados com uma resolução de 128x128 pixels, então eles podem perder um pouco de detalhes se você preferir ícones realmente grandes (por padrão os ícones são mostrados em uma resolução de 64x64 pixels). Tenha certeza de ter exportado seus dados ou criado um ponto de restauração primeiro, assim está ação poderá ser revertida. Há também a opção de converter cada ícone individualmente no menu do favorito.

Restore Points

Pontos de Restauração

Here you can create restore points. They are useful if you want to revert changes or just look at old bookmarks you removed. They are created automatically into the time frame of your choice or manually. You can also choose the maximum number of restore points that will be created so, if that number is reached, new restore points will replace the older ones.
These points are created only on this device and are not be affected by resets or imports. They are not synced with other devices or exported either and will be removed if you uninstall this application.
You can preview, delete or download them. Previews can be browsed but not changed. If you choose to restore some point remember that the current data will be lost and, if your synchronization is active, the sync data will be overwrited, so disconnect from Dropbox if you don't want this to happen. Aqui você pode criar pontos de restauração. Eles são úteis caso você necessite reverter alguma mudança ou apenas procurar favoritos antigos que você removeu. Eles são criados automaticamente na janela de tempo à sua escolha ou manualmente. Você também pode escolher o número máximo de pontos de restauração que serão criados assim, caso este número for alcançado, novos pontos sobrescreverão os mais antigos.
Estes pontos são criados apenas neste dispositivo e não são afetados por redefinições ou importações. Eles também não são sincronizados com outros dispositivos ou exportados e serão removidos caso você desinstale esta aplicação.
Você pode visualizar, remover ou baixar cada um deles. Visualizações podem ser navegadas mas não modificadas. Caso você escolha restaurar um ponto, lembre-se de que seus dados atuais serão perdidos e, caso a sincronia esteja ativa, os dados se sincronia serão sobrescritos, então, desconecte do Dropbox se você não quiser que isso aconteça.

Synchronization

Sincronização

Dropbox

You can connect to your Dropbox account to synchronize your data with others devices. Once the first connection is stablished, a sync file is created with the current data. However, if a sync file is found at Dropbox it WILL OVERWRITE the current data. You can delete the Dropbox sync file located at your Dropbox Apps folder if you want to start a brand new sincronization with the current data. To start the process follow the steps below:

Você pode conectar a sua conta do Dropbox para sincronizar seus dados com outros dispositivos. Uma vez que a conexão é estabelecida, um arquivo de sincronia será criado com os dados atuais. Porém, caso um arquivo de sincronia seja encontrado no Dropbox, ele SOBRESCREVERÁ os dados atuais. Você pode remover o arquivo de sincronia localizado na pasta de Aplicações do Dropbox se quiser iniciar uma nova sincronia com os dados atuais. Para iniciar o processo siga os passos abaixo:

  1. Get an authorization code Obtenha um código de autorização
    Get Authorization Code Obter Código de Autorização
  2. Paste the code in the field below and click the "Save the Code" button Cole o código no campo abaixo e clique no botão "Salvar o Código"
Dropbox Connected Dropbox Conectado

Your data is now synced through your Dropbox account. The synchronization file can be found inside you Dropbox Apps folder.

Seus dados agora estão sincronizados através da sua conta do Dropbox. O arquivo de sincronia pode ser encontrado dentro da sua pasta de Aplicações do Dropbox
